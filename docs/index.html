Digital Commodity Ecosystem

Ekosistem GOAT dan MEAT menghubungkan dunia pertanian dengan blockchain melalui tokenisasi siklus hidup hewan ternak, menciptakan sistem nilai yang transparan dan dapat diaudit.

GOAT Token

Guardian of Agricultural Trade - Token utilitas yang mendukung proses staking dan mencerminkan nilai ternak hidup.

  • Dicetak saat NFT kambing dikunci (wrapped)
  • Dapat di-stake untuk memperoleh imbal hasil
  • Mekanisme klaim dan kompaun fleksibel

MEAT Token

Market-Enabled Agricultural Token - Token yang mewakili daging fisik dari ternak yang disembelih.

  • Dicetak saat NFT ternak dibakar
  • Mewakili subtype spesifik (GOATMEAT, BEEFMEAT)
  • Dapat ditebus menjadi produk fisik

Lifecycle NFT

NFT yang mewakili identitas digital hewan ternak sepanjang siklus hidupnya.

  • Mencatat data ternak (NFC ID, ras, berat)
  • Dapat diperbarui dan ditransfer
  • Dibakar saat penyembelihan untuk mencetak MEAT

Token Lifecycle Flow

Alur lengkap dari kelahiran hewan ternak hingga penebusan daging fisik melalui token digital.

1. Mint GoatNFT

Setiap kambing yang lahir dicetak sebagai NFT unik dengan data identitas dan berat awal.

2. Update Weight

Pemilik memperbarui berat kambing secara berkala untuk menjaga keakuratan valuasi.

3. Wrap & Stake

Kunci GoatNFT untuk mencetak token GOAT yang dapat di-stake untuk memperoleh reward.

4. Burn NFT

Saat penyembelihan, NFT dibakar dan mencetak MEAT sesuai berat terakhir.

5. Redeem MEAT

Token MEAT ditebus menjadi produk daging fisik melalui RedeemEngine.

Technical Implementation

Implementasi kontrak pintar yang mengatur seluruh ekosistem token dan alur kerja.

GoatNFTWrapper

Kontrak yang mengunci GoatNFT dan mencetak token GOAT sesuai berat kambing.

Solidity Implementation 
function wrap(uint256 tokenId) external {
    require(goatNFT.ownerOf(tokenId) == msg.sender, "Not owner");
    goatNFT.transferFrom(msg.sender, address(this), tokenId);
    
    uint256 weight = goatNFT.getWeight(tokenId);
    uint256 amount = weight * GOAT_WRAP_RATE;
    
    goatToken.mint(msg.sender, amount);
    emit Wrapped(msg.sender, tokenId, amount);
}

Burn Hook

Mekanisme otomatis yang mencetak MEAT saat NFT dibakar.

Burn Function 
function burn(uint256 tokenId) external {
    require(_isApprovedOrOwner(msg.sender, tokenId), "Not owner");
    
    uint256 weight = goatData[tokenId].weight;
    _burn(tokenId);
    
    uint256 meatAmount = (weight * SLAUGHTER_YIELD_BPS) / 10000;
    bytes32 subtype = encodeBytes32String("GOATMEAT");
    
    meatToken.mintSubtype(msg.sender, subtype, meatAmount);
    emit GoatBurned(tokenId, weight);
}

Subtype MEAT Structure

Token MEAT mendukung berbagai subtype yang mewakili jenis daging berbeda.
